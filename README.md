# ClustIRR_melanoma_paper_data
Scripts and processed datasets from the paper "Spatial Compartmentalization of TCR Repertoires Between Primary 
Melanomas and Sentinel Lymph Nodes Reveals Distinct Clonal Architectures and Shared Antigen Recognition"

## Datasets 
directory **data/**: processed TCRbeta repertoires

## Code to reproduce analyses and recreate figures
directory **src/**: Code to generate main figures 1-7 and the supplementary files
